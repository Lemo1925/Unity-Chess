初始化棋盘：

直接在预制件里放全部点位的transform然后放到一个list里
写一个棋盘的Mono脚本挂到棋盘预制件上就行然后棋盘Mono里有一个list[transform]
直接在预制件里放一堆空gameobject，然后移到对应点上，在把这些transform再拖到棋盘Mono上的list就好了

private float 
    X = Common.BoardSize.x, 
    Y = Common.BoardSize.y;
    
 for (int i = 0; i < 8; i++)
        {
            if (ChessCamp == Camp.WHITE)
            {
                var pawn = Instantiate(
                    PawnPrefab, 
                    new Vector3(1.125f + 2 * X, 0, 7.375f - i * Y),
                    Quaternion.identity);
                Pawns.Add(pawn);
            }
        }
